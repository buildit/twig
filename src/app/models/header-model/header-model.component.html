A Model serves as the foundation of a Twiglet. Each Twiglet must have a model associated with it. The model determines the type of entities used to depict people, places, or things in Twiglet nodes. It also sets the icon and color used to depict the entity.

Model Dropdown - Allows access to pre-existing models.

  • Clone - Creates a copy of the model, including its entities and entity attributes.
  • Rename - Overwrites the title of the model.
  • Delete - Discards the Model, but doesnâ€™t affect Twiglets using it.

Model View Mode allows an at-a-glance look at the entity types in a model. It shows the Type, icon, and color used to represent each entity.

  • Entity - Object that makes up a node in a Twiglet. May be a person, place, or thing.
  • Attributes - provide more detail about a specific entity and may be designated as required or optional when adding nodes.
